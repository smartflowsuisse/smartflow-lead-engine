# Construction Leads Import Guide

Goal: prepare clean Construction / Bau / Renovation leads for SmartFlow Lead Engine outreach.

Required CSV columns:
company,website,email,phone,city,industry,notes

Good lead criteria:
- active website
- public business email
- Swiss city
- construction, renovation, metallbau, holzbau, electricite, sanitaire, or chauffage activity
- likely invoices, supplier documents, project follow-ups, reporting, or procurement
- not already contacted in CRM
- not Won or Lost

Import workflow:
1. Open Import CSV.
2. Paste CSV with required headers.
3. Run Preview.
4. Check invalid rows and duplicates.
5. Import only valid rows.
6. Open Templates.
7. Select Construction Automation Starter.
8. Open Outreach.
9. Use Show matched only.
10. Test 2-3 email drafts before real sending.

Safety rule: do not import fake example companies.
