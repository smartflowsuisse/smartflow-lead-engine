### 2. Prepare audit checklist

The audit should check:

- website contact form;
- email visibility;
- response process;
- follow-up process;
- document handling;
- repetitive admin tasks;
- CRM or spreadsheet usage;
- appointment/call process;
- data-safety risks;
- automation opportunities.

### 3. Prepare data-safety rules

Rules:

- no client data is sent to AI without review;
- no automated email sending without approval;
- no access deletion or risky changes without confirmation;
- backups before important data changes;
- client tools stay simple;
- human remains responsible for final communication.

### 4. Prepare reply handling process

Possible reply statuses:

- interested;
- asks for price;
- asks for details;
- wants a call;
- not interested;
- no response;
- unclear reply.

CRM flow:

Contacted → Replied → Meeting → Proposal → Won / Lost.
