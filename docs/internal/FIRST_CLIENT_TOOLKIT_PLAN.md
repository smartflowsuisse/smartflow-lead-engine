## 2. Main rule for the next month

Do not focus on sending more emails.

Focus on building the client-ready toolset:

- clear offer;
- audit checklist;
- PDF materials;
- Make demo;
- Google Sheets CRM demo;
- reply handling workflow;
- data safety rules;
- onboarding process.

AI can prepare drafts, but a human must review and send manually.

## 3. Role of each tool

| Tool | Role |
|---|---|
| SmartFlow Lead Engine | Internal lead discovery, analysis, CRM, outreach preparation |
| Make | Main automation demo and client workflow builder |
| Google Sheets | Simple CRM/demo table for client workflows |
| Google Drive / Docs | Internal documents, checklists, offers, process storage |
| Canva | PDF offer, mini-audit, commercial proposal, visual diagrams |
| Gmail / Calendar | Manual client communication and call scheduling |
| AI tools | Drafting, analysis, website audit, email preparation |
| Screen recording / Loom / Zoom | Short client demo video |
| SmartFlow website | Trust layer and business presentation |

## 4. What not to build now

Do not build or connect these now:

- public SaaS;
- bulk email automation;
- LinkedIn automation;
- HubSpot;
- expensive CRM;
- Stripe;
- complex BI dashboard;
- n8n before Make is validated;
- Airtable before Google Sheets is enough;
- Figma before Canva is enough;
- Notion before Google Docs is enough.
