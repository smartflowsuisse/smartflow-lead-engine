## Simulated lead review

Observed problems:

- website does not clearly explain the renovation workflow;
- contact form exists but there is no visible follow-up process;
- no clear response time is shown;
- no structured project intake form;
- no simple way to qualify request type, urgency, location, or budget range;
- no visible automation for follow-up or internal task assignment.

Potential quick wins:

1. Add structured request form fields.
2. Create automatic internal notification for new requests.
3. Prepare AI-assisted draft reply for manual review.
4. Add CRM tracking for lead status and next action.
5. Add basic follow-up reminder after 2 business days.

## One-page audit outline

Audit title: Améliorer le suivi des demandes clients

Main finding:

The company receives requests through different channels, but the intake and follow-up process is not structured enough. This can create missed opportunities, delayed replies, and unclear internal responsibility.

Recommended improvements:

1. Standardize the request form.
2. Send each new request into a simple CRM table.
3. Generate an internal AI summary of the request.
4. Prepare a draft reply for manual validation.
5. Add a follow-up reminder if no answer is sent within 2 business days.

Expected benefit:

Better visibility, fewer forgotten requests, faster manual response preparation, and a clearer workflow for the team.

Important note:

No fixed savings are promised. Any time savings must be verified through real workflow analysis.
