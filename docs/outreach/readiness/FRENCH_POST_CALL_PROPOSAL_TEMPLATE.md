# French Post-Call Proposal Template — SmartFlow Suisse

Purpose: internal template for sending a short commercial proposal after a discovery call.

Status: internal only. Review and adapt before sending.

---

## 1. Email structure

Subject: Proposition — amélioration ciblée de votre site / parcours de contact

Bonjour [Nom],

Merci pour notre échange.

Suite à notre discussion, je vous propose une première intervention ciblée, avec un périmètre limité et clair, afin d’améliorer [objectif principal] sans partir sur une refonte complète.

---

## 2. Context

D’après notre échange, les priorités principales sont :

- [Priorité 1]
- [Priorité 2]
- [Priorité 3]

Le site actuel dispose déjà d’une base solide. L’objectif serait donc de renforcer certains points précis, notamment :

- l’expérience mobile ;
- le parcours de contact ;
- la clarté des demandes entrantes ;
- les points techniques essentiels.

---

## 3. Proposed scope

### SmartFlow Quick Win

L’intervention comprend :

1. Analyse rapide de l’existant
   - site ;
   - version mobile ;
   - parcours de contact ;
   - points techniques prioritaires.

2. Améliorations ciblées
   - optimisation de certains éléments techniques ;
   - amélioration du parcours de contact ;
   - ajustements simples sur mobile ;
   - vérification du formulaire ou des liens de contact si applicable.

3. Résumé avant / après
   - points observés ;
   - actions réalisées ;
   - recommandations pour la suite.

---

## 4. Exclusions

Cette proposition ne comprend pas :

- refonte complète du site ;
- nouveau branding ;
- campagne SEO complète ;
- publicité Google Ads ;
- développement CRM complet ;
- automatisation avancée ;
- garantie de volume de leads ;
- garantie de positionnement Google.

---

## 5. Price

Option recommandée :

SmartFlow Quick Win  
Prix : 690 CHF  
Durée estimée : 2 à 5 jours ouvrables

Ce prix suppose :

- un périmètre limité ;
- un accès technique disponible ;

- aucune intégration complexe ;
- une intervention techniquement faisable après vérification.

If scope is larger:

Si le périmètre s’avère plus large après vérification technique, je vous le signalerai avant toute intervention. Aucun travail supplémentaire ne sera réalisé sans validation.

Si cette approche vous convient, la prochaine étape serait :

## 8. Internal adaptation checklist

Before sending, adapt:

- client name;
- company name;
- main pain point;
- exact website issue;
- price;
- timeline;
- exclusions;
- next step;
- whether access is needed;
- whether call summary is accurate.

---

## 9. Internal risk check before sending

Do not send if:

- scope is unclear;
- website technology is unknown;
- access is impossible;
- client expects full redesign;
- client expects guaranteed leads;
- project seems too large for first client;
- legal/data-sensitive automation is involved.
