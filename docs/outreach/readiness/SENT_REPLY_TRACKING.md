Purpose: internal tracking note for manual outreach only.

## Rules

- Do not record a lead as sent before the email is actually sent manually.
- Do not mark a lead as Contacted before sending.
- Do not automate sending.
- Record every manual send.
- Record every reply.
- Use prepared templates before answering.
- If the client asks for price, use the pricing guide.
- If the client asks for a call, use the discovery call checklist.
- If the client is interested, send the mini-audit before pushing a call.

## Internal response workflow

If reply is positive:
1. Send mini-audit.
2. Keep message short.
3. Offer call only after delivering value.

If client asks price:
1. Use FIRST_CLIENT_PRICING_AND_OFFER_GUIDE.md.
2. Give range.
3. Avoid fixed price before scope is clear.

If client asks for a call:
1. Use FRENCH_DISCOVERY_CALL_CHECKLIST.md.
2. Prepare 2–3 time slots.
3. Define call objective.

If client refuses:
1. Reply politely.
2. Do not push.
3. Do not follow up again.

If no reply:
1. Wait.
2. One polite follow-up maximum.
3. Do not start aggressive sequence.
